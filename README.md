# うんこ君バトルフィールド

`index.html` をブラウザで開けば遊べます。iPadで動かすなら、このフォルダを静的配信してSafariで開くのが安定です。

簡単なローカル起動例:

```bash
python3 -m http.server 8000
```

その後 `http://localhost:8000` を開きます。

## いま入っている要素

- 左右移動、ジャンプ、うんこ投げ
- 敵の色で難易度が変わる
- 暗い敵ほど速く、投げる間隔が短く、HPが高い
- うんこ命中時の爆発エフェクト
- 状態ごとに変わる簡易スプライト
- タッチ操作とキーボード操作
- Web Audio API の簡単なBGMと効果音

## 次に自然言語で変えやすい場所

- 敵の強さ:
  `game.js` の `difficultyPalette`
- プレイヤー性能:
  `game.js` の `config`
- WAVEごとの敵数:
  `game.js` の `spawnWave`
- 見た目や配色:
  `styles.css`
- 画面内テキスト:
  `index.html` と `game.js`

## 変更依頼の例

- 「3WAVEごとにボスを出して」
- 「敵の弾をもっと遅くして」
- 「当たっても2回まで耐えられるモードを作って」
- 「背景を森じゃなくて宇宙にして」
- 「うんこの代わりに星を投げる版を作って」
